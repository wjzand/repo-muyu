## 1. 产品概述

赛博木鱼是一款融合赛博朋克美学与传统修行元素的H5网页工具。用户通过点击屏幕模拟敲木鱼，积累赛博功德值，体验沉浸式的数字修行。内置多种电子音色、霓虹灯光效与动态经文，支持敲击计数、自动敲击、成就解锁与排行榜比拼，让现代人在数字世界找到片刻宁静，或与好友进行轻松搞怪的"功德竞赛"。

- 核心目标：打造一款高沉浸感、视觉冲击力强的赛博禅意H5应用
- 目标用户：追求趣味解压的年轻网民、赛博朋克文化爱好者、程序员群体

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 游客 | 无需注册 | 体验基础敲击、功德积累 |
| 普通用户 | 本地存储自动生成 | 全部功能，含排行榜、皮肤、成就 |

### 2.2 功能模块
1. **修行主界面**：木鱼敲击、音效震动、粒子波纹、自动敲击、快捷功能栏
2. **功德体系页**：功德值展示、修行等级、升级动画、每日上限
3. **经文系统**：浮动赛博经文、自定义经文、显示设置
4. **皮肤系统**：多款木鱼外观、解锁条件、切换动画、专属音效
5. **排行榜页**：本日榜/本周榜/总榜、好友PK、修行海报生成
6. **个人中心**：用户信息、成就徽章、设置项、数据管理

## 3. 核心流程

**主流程**：用户进入首页 → 点击木鱼敲击 → 积累功德 → 解锁等级/皮肤/成就 → 查看排行榜 → 分享海报

**等级升级流程**：敲击积累功德 → 达到阈值 → 触发升级动画 → 解锁新称号/皮肤 → 提示可解锁内容

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：霓虹紫(#B026FF) + 赛博青(#00F0FF) + 霓虹粉(#FF2D95)
- **辅助色**：暗黑底色(#0A0A1A)、深紫渐变(#1A0A2E)
- **按钮风格**：霓虹发光边框，hover时亮度增强，点击时闪烁
- **字体**：标题使用像素/故障风格字体，正文使用科技感等宽字体
- **布局**：移动端优先，木鱼居中占据主视觉区域
- **图标风格**：线条图标配合霓虹发光效果

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 修行主界面 | 木鱼区域 | 3D木鱼居中，霓虹描边，点击缩放动画，径向波纹扩散 |
| 修行主界面 | 计数显示 | 故障艺术字体，功德数字滚动增长，高亮闪烁 |
| 修行主界面 | 功能栏 | 底部半透明工具栏，霓虹图标按钮 |
| 修行主界面 | 背景 | 深色渐变+星空粒子+漂浮光点 |
| 排行榜 | 榜单列表 | 霓虹边框卡片，前三名特殊样式 |
| 个人中心 | 用户卡片 | 赛博风格头像框，霓虹进度条 |

### 4.3 响应式设计
- 移动端优先设计，木鱼敲击区域占屏幕主要面积
- 支持竖屏与横屏自适应
- PC端适配：键盘空格键敲击，更大屏幕展示更多视觉细节
- 触摸优化：木鱼区域足够大避免误触

### 4.4 动效设计
- 敲击：木鱼放大缩小+径向波纹扩散
- 功德值：数字滚动变化，增长时高亮闪烁
- 升级：全屏霓虹光芒汇聚+等级称号弹出
- 经文：从屏幕边缘飘入，渐隐消失
- 皮肤切换：故障艺术过渡动画
